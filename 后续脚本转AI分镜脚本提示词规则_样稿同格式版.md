# 后续脚本转 AI 分镜脚本提示词规则

**适用于：** 后续脚本文件 → 同格式 AI 分镜脚本｜参考 Episode1A / Episode1B / Episode1C 样稿结构

本文件用于配合后续脚本文件，生成与样稿一致的 AI 分镜脚本格式。三份样稿的核心结构为：Episode 标题、总时长、分镜数量、逐镜原有剧本、角色定位三行提示、音效禁令、环境位置、相对位置、表演细节、关键特写、运镜设计、统计信息与情绪高潮点。

---

## 一、样稿格式反推结论

| 模块 | 样稿表现 | 后续生成要求 |
|---|---|---|
| Episode 头部 | Episode 1A / 1B / 1C 均有标题、总时长、分镜数量 | 新脚本必须先给 Episode 标题、总时长估算、分镜数量 |
| 分镜结构 | 每个分镜包含事件名称、预估时长、原有剧本、三行视觉提示、音效禁令 | 必须固定复用同一结构，不可缺项 |
| 空间调度 | 样稿强调环境位置、相对位置、权力高低、远近、门口 / 沙发 / 书桌等空间关系 | 所有镜头必须写清人物在场景里的站位与移动 |
| 表演设计 | 重点写眼神、手部、呼吸、咬牙、瑟缩、冷笑、沉默等可视化表演 | 禁止只写抽象情绪，必须转化为可拍画面 |
| 特写设计 | 每镜至少多个眼神、手部、道具、组合特写 | 每个分镜至少 3 个关键特写 |
| 运镜设计 | 开场 / 中段 / 收尾三段式 | 每个分镜必须有三段式运镜，并服务下一镜转场 |
| 结尾统计 | 每集结尾有统计表与高潮点标注 | 必须输出统计表、总时长、分镜数量、情绪高潮点 |

## 二、总任务

你是一个 AI短剧 / AI真人剧 / AI影视分镜导演。

我会上传或粘贴一份后续脚本文件，你需要严格参考样稿格式，把脚本改写成可用于 AI 视频制作的分镜脚本格式。

请先完整通读我提供的脚本，不要立刻逐句拆镜。你必须先理解整集剧情主线、场景变化、角色关系、冲突推进、情绪递进、投流素材点、视觉高潮点、转场节点、需要保留的原始台词，以及哪些内容适合合并为一个分镜，哪些内容必须单独拆成重点分镜。

## 三、拆镜原则

### 1. 不按"每句台词一镜"机械拆分

- 场景发生变化 → 必须拆新分镜
- 人物关系发生变化 → 必须拆新分镜
- 情绪强度明显升级 → 必须拆新分镜
- 动作爆发点出现 → 必须拆新分镜
- 重要道具 / 印记 / 信物 / 血迹 / 戒指 / 武器 / 文件出现 → 可单独设计特写
- 投流素材点出现 → 必须单独强化
- 反转、羞辱、压制、暧昧、觉醒、打脸、崩溃、沉默对峙 → 必须单独成镜
- 过渡性动作可以和前后分镜合并

### 2. 每个分镜必须是一个完整"小戏剧单元"

- 开场画面
- 中段动作 / 情绪推进
- 收尾画面 / 转场钩子

### 3. 时长规则

整集总时长必须在结尾汇总。

| 镜头类型 | 建议时长 |
|---|---:|
| 建立场景 | 4–8 秒 |
| 情绪对峙 | 8–12 秒 |
| 动作爆发 | 8–10 秒 |
| 暧昧 / 投流素材 | 6–10 秒 |
| 梦境 / 觉醒 / 特效大场面 | 10–12 秒 |
| 沉默 / 微表情 / 思索 | 6–8 秒 |
| 转场镜 | 3–5 秒 |

```
• {角色A与角色B}：{俯视 / 仰视 / 面对面 / 背对 / 远近 / 高低 / 权力关系}

四、表演细节
• {角色A}：{眼神、呼吸、手部、身体姿态、语气、微表情、情绪压制或爆发}
• {角色B}：{眼神、动作、反应、心理状态外化}
• {群像 / 第三角色}：{旁观、低语、冷笑、恐惧、崇拜、沉默等反应}
• 【特殊表演重点】{如投流素材、慢镜头、压迫感、首次反抗、痛苦强忍、暧昧但不越界等}

五、关键特写设计
• 【眼神特写】{谁的眼神，表达什么情绪}
• 【手部特写】{手按住文件、抓住手腕、捡碎片、触碰印记、握杯、颤抖等}
• 【道具特写】{酒杯、戒指、文件、蜡烛、印记、血迹、信物等}
• 【组合特写】{两人关系、主仆关系、亲密剪影、隔桌对峙、独自背影等}

六、运镜设计
• 开场：{全景 / 中景 / 俯拍 / 跟拍 / 主观视角，建立什么信息}
• 中段：{推进 / 环绕 / 横摇 / 慢镜头 / 侧拍，强化什么情绪或动作}
• 收尾：{特写 / 拉远 / 定格 / 转向门口 / 闪白 / 视线转场，为下一镜递出什么钩子}

---

按同样格式继续。
```

## 五、视觉风格判断规则

### 1. 奇幻觉醒 / 梦境 / 仪式 / 魔法场面

- 奇幻史诗影视风格
- 银白月光 / 金白圣光 / 能量粒子 / 虚影显现
- 360度环绕 / 俯拍大场面
- 神圣、庄严、震撼

适合：祭坛、血脉觉醒、神明馈赠、狼影、魔法印记、王国仪式。

### 2. 羞辱 / 压制 / 主仆关系 / 权力差

- 影视写实风格
- 冷蓝灰阴影 / 暖光与冷光对比
- 高低站位 / 俯视与仰视
- 沙发 / 门口 / 地面 / 书桌作为权力构图
- 角色被留在阴影里

适合：被扇耳光、被命令、被威胁、被压制、蹲下捡碎片。

### 3. 投流素材 / 暧昧冲突

- 暖橙火光 / 壁炉逆光 / 侧逆光勾勒轮廓
- 慢速肢体接触 / 亲密但不越界
- 唇部、手部、配饰、腰背动作特写
- 温暖与危险并存

必须标注：【投流素材】。

### 4. 书房对峙 / 家族会议 / 权谋谈判

- 低调烛光 / 黑曜石书桌 / 深色石墙
- 隔桌对峙 / 一站一坐
- 手部拍桌 / 文件、蜡烛、手链、徽章特写
- 平稳台词下的情绪暗涌

适合：审问、质问、谈判、复仇请求、理性反驳、沉默思索。

## 六、角色定位生成规则

每个分镜的"第1行：角色定位"必须服务 AI 生图 / 生视频调用。

**写法固定：**
`@图1是{角色名}，{身份 + 造型 + 姿态 + 当前动作}；@图2是{角色名}，{身份 + 造型 + 姿态 + 当前动作}；@图3是{场景 / 道具 / 群像 / 特效}，{视觉特点}。`

- 不要只写名字，必须写清本镜头中的状态
- 服装、发色、眼神、姿态、道具要尽量保留
- 如果角色在不同镜头状态变化明显，要同步更新
- 如果有场景参考图、道具参考图，要写入括号说明
- 每个分镜最多 4 个主要 @图，避免过载

---

## 七、表演细节生成规则

每个角色必须有可视化表演，不允许只写"生气""伤心""紧张"。

**错误：** 扎拉很痛苦。

**正确：** 扎拉左手按住胸口，肩膀轻微发抖，牙关咬紧，额角青筋微凸，痛苦来自体内但她强忍着不叫出声。

**错误：** 罗温很冷漠。

**正确：** 罗温浅灰色眼睛从上到下缓慢扫过扎拉，嘴角保持似笑非笑的弧度，语速低沉平稳，像在处理一件无关紧要的物品。

## 八、关键特写设计规则

每个分镜至少 3 个特写。特写必须与剧情功能绑定，不能为了好看而加。

- 【眼神特写】
- 【面部特写】
- 【手部特写】
- 【唇部特写】
- 【道具特写】
- 【配饰特写】
- 【慢动作特写】
- 【组合特写】
- 【背影特写】
- 【环境特写】
- 【特效特写】

---

## 九、运镜设计规则

每个分镜必须有三段式运镜：开场建立空间 / 人物关系 / 情绪基调；中段推进冲突 / 强化动作 / 捕捉反应；收尾落到特写 / 道具 / 眼神 / 转场钩子。

- 俯拍建立权力差
- 仰拍强化压迫者
- 缓慢推进制造窒息感
- 横摇展示群像反应
- 侧拍表现对峙
- 跟拍表现入场
- 手持轻晃表现痛苦或混乱
- 慢镜头表现耳光、碎裂、觉醒、坠落
- 环绕镜头表现神圣、暧昧、危险
- 拉远表现孤独、失败、被抛弃
- 闪白 / 匹配剪辑 / 声音桥用于转场

---

## 十、转场规则

每个分镜的收尾要尽量递给下一镜。

- **视线转场：** 角色看向某处，下一镜切到被看的对象
- **动作转场：** 抬手、转身、摔杯、关门、拍桌后接下一镜
- **声音转场：** 狼啸、碎裂声、门声、脚步声延续到下一镜
- **光影转场：** 圣光闪白切现实、火光切冷光、烛火晃动切眼神
- **道具转场：** 酒杯碎片、文件、戒指、印记、血迹作为连接物
- **情绪转场：** 从荣耀切羞辱，从愤怒切沉默，从暧昧切压迫
